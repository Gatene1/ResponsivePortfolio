About This Project

Blink is a top-down puzzle game about navigating minimal mazes to reunite with a mysterious character on the other side. Currently in development, its focus is on movement-based problem solving and level-based progression.

Core Features

  • Find the princess-style game
  • One-screen puzzle design
  • Pixel-perfect pathfinding

Built With

  • JavaScript + HTML5 Canvas
  • Custom engine with typed array optimizations
  • Modular level editor system

Status: In active development
